# Icon 图标

# 使用方法
直接通过设置类名来使用即可。例如：

:::demo
```vue
<template>
  <div class="demo-icon">
    <i class="i-ic-baseline-edit"></i>
    <i class="i-ic-baseline-share"></i>
    <i class="i-ic-baseline-delete"></i>
  <div>
</template>
```
<style>
.demo-icon > i {
  width: 24px;
  height: 24px;
  margin-right: 20px;
}
</style>
:::
